# Smart Wealth Dashboard - Interactive Web Elements 🚀

## 📖 Project Description
This is a dynamic, interactive web application built as part of the DecodeLabs industrial training program (Project 3). The core objective of this project is to implement JavaScript to manipulate the Document Object Model (DOM) and engineer a responsive "client-side nervous system". 

It acts as a premium dashboard for business and IT professionals to calculate ROI, Crypto returns, and Business Taxes seamlessly on a single page.

**Key Features & Engineering Standards:**
* **The IPO Loop:** Fully implements the Input-Process-Output loop for all mathematical calculations.
* **State Management:** Features a functioning Dark Mode toggle that manages visual states across the entire application.
* **Dynamic Content:** Utilizes JavaScript to instantly switch between different calculator forms (Tabbed Interface) without reloading the page.
* **Decoupled Architecture:** Strictly adheres to professional standards by using `js-` prefixes exclusively for JavaScript event hooks, and `is-` prefixes (e.g., `.is-active`, `.is-profitable`) strictly for visual state changes in CSS.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and decoupled hooks.
* **CSS3:** Advanced styling, CSS variables for theming, and smooth transitions.
* **Vanilla JavaScript:** DOM manipulation, Event Listeners, and core business logic.

## 🚀 How to Run the Project
1. Download or clone this repository to your local machine.
2. Navigate to the project folder.
3. Simply double-click the `index.html` file to open it in any modern web browser.
4. Test the interactivity by toggling Dark Mode and switching between the calculator tabs.
